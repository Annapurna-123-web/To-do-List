To-Do List Web Application

# Overview
A simple and interactive web application that helps users manage daily tasks efficiently using a clean and responsive interface.

# Features
- Add new tasks
- Edit existing tasks
- Delete tasks
- Real-time updates using JavaScript
- Dynamic DOM manipulation

# Tech Stack
- HTML
- CSS
- JavaScript

# How It Works
- Users can input tasks through the UI
- Tasks are dynamically added to the DOM
- JavaScript handles all CRUD operations
- UI updates instantly without page reload
